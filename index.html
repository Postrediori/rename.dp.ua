Полный перечень переименований улиц Днепропетровска по районам

Все названия на украинском языке согласно оригиналу распоряжения о переименовании. Если объект переименования расположен на территории двух и более районов - то он будет присутствовать в списке каждого района.

Весь массив данных по переименованиям доступен для скачивания и использования в формате JSON по этой ссылке. Коды районов в этом файле ("r61", "r62", ... ) взяты исходя из существующей системы кодировки районов, которая используется органами исполнительной власти, для совместимости и облегчения экспорта данных.

Ведутся работы по накоплению и выдаче здесь информации о лицах, в честь кого произведено переименование.
